Settings — every physiological number is editable (your request from today): max HR, resting HR, optional lactate threshold, zone method, or fully custom bpm bounds per zone — and weight via weigh-ins anywhere (check-in, Progress, here). Change any of them and every bpm range, plan target and chart recomputes. Data section: export / import JSON, Garmin CSV import, reset behind a confirm; a quiet banner appears if you haven't exported in 30 days.

Palette — dark only

bg · screen · card · glacier cyan · ride · target sand
Z1 → Z5, one cool→hot ramp used everywhere

Run and ride are told apart by icon + hue (cyan vs periwinkle), never by competing palettes. Zone colors are reserved for intensity only.

Type — numbers are the heroes

45 min · 110–128 bpm
Eyebrow labels · letter-spaced caps

System font stack (offline constraint), compensated with a hard scale: 84 px duration, 38 px bpm, 11 px eyebrows. Tabular numerals throughout. Every tap target ≥ 44 px.

Motion & tone

One orchestrated moment: the check-in proposal slides up and the durations re-compute live under the slider (try it on frame 4). Everything else is instant. prefers-reduced-motion honored.

Copy never shames: "skipped — gone, no debt", "pick up where you left off". The locked-intervals card shows progress toward unlocking, not a padlock.
